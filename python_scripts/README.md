
Перед запуском нужно установить недостающие либы:

```
python -m pip install -r requirements.txt
```

В каталоге results лежат пример запуска данного скрипта - переданные параметры и файл результата выполнения.

Запуск осуществляется командой.

```
python -m use_cli parameters.json
```

По умолчанию прога работает с каталогом results, если не указываются абсолютные пути параметрам и результату.
Имя файле результата можно указать в parameters.json или же будет использоваться имя по умолчанию result-<тек-дата-время>.json
