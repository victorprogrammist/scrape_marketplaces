## Обзор
Здесь представлен сервис по парсингу маркетплейса Ozon.\
В будущем добавятся и другие маркетплейсы, и функционал будет развиваться.\
Точнее, здесь представлены программы-клиенты, которые демонстрируют возможности сервиса и способы подключения.
На текущий момент сервис в тестовом режиме и бесплатен.

## Использование из 1С

Конфигурация 1с - юмп.dt (название наверное от use_marketplaces), которая подключается к сервису сканирования маркетплейсов.
Подключается и по указанным отборам загружает товары и цены.

Сама конфигурация 1с довольно таки простая, и представляет демонстрацию как загружать товары из сервиса.

## Использование из других скриптов (python)

В папке python_scripts пример использования на питоне.
В частности в папке python_scripts/results пример выполнения команды:
```
python -m use_cli parameters.json
```

## Технические детали подключения
Для подключения к сервису не используется выделенный домен или статический IP.
Вместо этого, есть временный домен, который записывается в файл на GitHub.
И 1С, и python_script, сначала считывают этот динамическое имя, и затем уже по нему обращаются к сервису.
Причины такой связки, отчасти, что сам сервис по сканированию МаркетПлейсов не может быть на статическом IP,
иначе его маркетплейсы быстро блокируют. Позже, когда и если сервис перейдет из тестового режима в коммерческий режим,
то приделаю более сложную сетевую развязку, которая позволит обращаться непосредственно по статистическому IP.\
\
Отсылаемый запрос и получаемый ответ всегда запакован - zlib,gzip,zipfile.\
Сервер отвечает в том же варианте упаковки, в каком упакован запрос.\
Для подключения используются заголовки (что можно увидеть в python_scripts\serverless_client\slc.py):
```
    "Bypass-Tunnel-Reminder": "true",
    "Content-Type": "application/octet-stream",
    "Content-Transfer-Encoding": "binary",
    "Content-Length": str(len(packed_request))
```

## Форматы данных
Запрос формируется в виде JSON текста.\

Поля доступные к указанию в запросе:

| Ключевое поле | Описание |
| :--- | :--- |
| `search_string` | Строка поиска. Если не указано, парсятся все товары из категории |
| `url_category` | Ссылка на категорию. Если не указано, поиск идет с главной страницы |
| `expected_count_items` | Количество возвращаемых товаров (ограничено 1000) |
| `min_price` | Минимальная цена для фильтрации (опционально) |
| `max_price` | Максимальная цена для фильтрации (опционально) |
| `sorting` | Тип сортировки (значение точно как на сайте, например: `"Популярное"`) |


Доступные данные в результатном JSON:

| Поле | Описание |
| :--- | :--- |
| `sku` | Артикул товара |
| `title` | Название товара |
| `url` | Ссылка на карточку товара |
| `badges` | Бейджики списком |
| `badge` | Бейджики одной строкой через запятые |
| `src_price` | Цена, как она отображена на сайте |
| `price` | Цена, отформатированная в число |
| `src_price_fake` | Цена «до скидки», как она отображена на сайте |
| `price_fake` | Цена «до скидки», отформатированная в число |
| `src_rating` | Рейтинг, как он отображен на сайте |
| `rating` | Рейтинг, отформатированный в число |
| `src_count_feedbacks` | Количество отзывов, как оно отображено на сайте |
| `reviewCount` | Количество отзывов, отформатированное в число |
| `brand` | Бренд товара |
| `src_residue` | Остаток товара, как он отображен на сайте |
| `residue` | Остаток товара, отформатированный в число |
| `delivery` | Доставка списком |
| `deliveryInfo` | Доставка одной строкой через запятые |
